# 🦈 מדריך לבוט העתקות פולימרקט (Polymarket Copy-Trading Bot)

ברוך הבא למערכת האנליטיקה והמעקב אחרי לוויתנים בפולימרקט! 
המערכת נוצרה כדי לאתר סוחרים רווחיים (עפ"י קטגוריות), לבצע עליהם סימולציית תאימות לסכומים קטנים (חישוב ROI וירטואלי) - ולאחר מכן לעקוב ולחקות אותם באמצעות "ארנק דמו" בזמן אמת.

---

## 🚀 איך מפעילים?
פתח את מסוף הפקודות (CMD) או את הטרמינל בתיקייה הזו והרץ את הפקודה:
```bash
python main.py
```
יפתח בפניך תפריט עם 3 אפשרויות:

### אפשרות 1: סורק ולוויתנים כלליים (Standard Scanner)
מושך את 5 הארנקים המרוויחים ביותר ביממה האחרונה בפולימרקט.
עובר על 100 העסקאות האחרונות של כל אחד מהם ומחשב כמה הבוט הוירטואלי שלנו היה מרוויח (או מפסיד) בהנחה וקנינו יחד איתם עם **עד 5 דולר** לכל טרייד.
> מצוין כדי ללמוד על סטטיסטיקות הרכישה היומיות שלהם (תדירות עסקאות, גודל עסקה ממוצע).

### אפשרות 2: התחלת מעקב בארנק דמו לייב (Live Paper Trading)
לאחר שבחרתם לעצמכם ארנק רווחי מהסורקים, בחרו באפשרות זו.
המערכת תשאל אתכם לכתובת הארנק (מתחיל ב-0x), ואז תיכנס ל**מצב מעקב אגרסיבי** בו היא סורקת את פולימרקט כל 2 שניות! ברגע שהמערכת תזהה שהלוויתן ביצע רכישה, היא תבצע קנייה אוטומטית (וירטואלית) מהתקציב שלכם (שמתחיל ב-50$).
> התיק נשמר ונשאר זמין בספריית המערכת תחת הקובץ `paper_portfolio.json`. אי-אפשר לרדת מ-0 דולר, פקודות רכישה נחתכות לפי היתרה בספר השנתי.

### אפשרות 3: סורק מתקדם וחישובי ROI מדויקים לפי קטגוריה (Advanced ROI)
מערכת מתוחכמת המאפשרת לחפש מומחים לפי סוג שוק ספציפי: `SPORTS` (ספורט), `CRYPTO` (קריפטו/ביטקוין) וכדומה.
המערכת משחזרת את כל הפוזיציות שארנק עשה לאחרונה, מחשבת את יחס ההפסד והרווח שלו באחוזים מדוייקים על מול החוקים שלנו (עד 5 דולר לעסקה), ומדרגת אותם לפי מי שהביא את ה- **ROI הגבוה ביותר**. 
> זו האפשרות המומלצת ביותר לבחירת מודל לחיקוי!

## 📁 הסבר על קבצי המערכת
כל קובץ אחראי על חלק אחר כדי שיהיה קל לקרוא את הקוד ולהוסיף לו יכולות בהמשך:

- `main.py`: **הקוד הראשי.** מציג את התפריטים למשתמש ומשלב את כל המערכות ביחד. מפה מתחילים לרוץ.
- `scanner.py`: מתחבר ללוח התוצאות של פולימרקט כדי לחלץ כתובת ארנקים (Proxy Wallets) של השחקנים שמרוויחים הכי הרבה ב-24 שעות האחרונות.
- `backtester.py`: מושך את העסקאות מהעבר של ארנק מסוים ומחשב (בעזרת הספרייה הפופולרית Pandas) נתוני בסיס וזמנים כמו "גודל עסקה ממוצעת" ואת הרווח המהיר לפי חוקי ה-5$. 
- `ledger_backtester.py`: (הגרסה המתקדמת של הבק-טסטר). ניגש היישר לתיק הפוזיציות של הארנק ומחשב באחוזי דיוק מקסימליים את שורת הרווח (ROI) האמיתית שהיית עומד לעשות עליו! משמש בעיקר את אופציה 3.
- `paper_trader.py`: **המוח של ארנקי הדמו.** זה הקובץ שמחזיק את הכסף הווירטואלי (50$) ויודע לייצר קניות ומכירות.
- `live_monitor.py`: פועל בלולאה מהירה מאד (כל 2 שניות מתשאל את ה-API), תופס עסקה ברגע שהיא קורית במציאות ומעביר אותה ישירות אל ה-paper_trader לביצוע סימולטור.
- `.env`: קובץ הגדרות. מכיל את המשתנים שיעזרו לנו בעתיד כמו URL לבלוקצ'יין, ואת הגדרות כמות הרכישה (5%).
